# Rubocop Mardkdown

Run Rubocop against your Markdown files to make sure that code examples follow style guidelines.


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rubocop-md'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubocop-md
